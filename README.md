## Aika AI

**Aika AI** is an AI-powered assistant currently being built to provide rich, conversational interactions backed by a modular backend (agents, prompts, storage, and LLM integrations). The goal of the project is to create a flexible, extensible assistant that can be adapted to different workflows and use cases.

---

### Backend quick start

- **Install dependencies**

```bash
cd backend
pip install -r requirements.txt
```

- **Configure environment**

Copy the example file and adjust values as needed:

```bash
cd backend
copy env.example .env   # on Windows (PowerShell / CMD)
```

- **Run the API server**

```bash
cd backend
uvicorn app.main:app --reload
```
